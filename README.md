## 필수적으로 필요한 모듈

- navigation 모듈 정의한후 컨테이너 정의가 꼭 필요합니다.

- 모듈화까지는 아니더라도 컴포넌트 분리해두고 사용할 수 있게끔 변경 처리 완료
